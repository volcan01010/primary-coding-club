How I became a software developer

I started programming seriously to understand this volcano.
We collected tens of thousands of temperature measurements from the infrared (thermal) camera. There were too many to analysis with a spreadsheet so we used MatLab.
When we plot the data we can see that the eruptions don't affect the fumarole temperatures.

Why I like it
